# GeoPandas Kerala Advanced

This project extends the first demo (**geopandas-kerala-demo**) by showcasing advanced GeoPandas workflows on Kerala district and road datasets. It covers spatial operations (clip, simplify, reproject, spatial index) and thematic visualization (choropleths with dummy attributes).

## Workflows Included
- Clip roads to district boundaries  
- Explode multipart geometries  
- Simplify district boundaries  
- Centroid with labels  
- Spatial Index for efficient road filtering  
- Choropleth Mapping using dummy attributes (Population, Literacy, Rainfall)

## Note
Population, literacy, and rainfall values are dummy data for demonstration. Replace with official Census of India or Kerala Open Data Portal statistics for real analysis.
